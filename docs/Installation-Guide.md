# Windows Server 2019 AD DS Installation Guide

## Requirements

- VMware Workstation
- Windows Server 2019
- Windows 10 Client
- 2 GB RAM minimum
- 40 GB Disk

## Installation Steps

1. Install Windows Server 2019.
2. Configure a static IP address.
3. Rename the server.
4. Install Active Directory Domain Services (AD DS).
5. Promote the server to a Domain Controller.
6. Create a new forest (LHAMD.ma).
7. Restart the server.
8. Create Organizational Units (OU).
9. Create Security Groups.
10. Create Domain Users.
11. Join the Windows 10 client to the domain.
12. Verify domain authentication.
