# Windows Server 2019 AD DS Lab Report

## Objective

Deploy and configure an Active Directory Domain Services environment using Windows Server 2019.

## Environment

- Hypervisor: VMware Workstation
- Server OS: Windows Server 2019
- Client OS: Windows 10
- Domain Name: LHAMD.ma
## Tasks Completed

- Installed Windows Server 2019
- Configured Static IP
- Renamed Server
- Installed AD DS Role
- Promoted Server to Domain Controller
- Created Organizational Units
- Created Security Groups
- Created Users
- Joined Windows 10 Client to Domain
- Verified Successful Login

- Users authenticated successfully
- Windows 10 joined to domain
- AD Users and Computers verified
